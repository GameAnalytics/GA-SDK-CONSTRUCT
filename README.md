GA-SDK-CONSTRUCT
==========

GameAnalytics Construct SDK.

Documentation can be found [here](https://gameanalytics.com/docs/construct-sdk).

> :information_source:
> The Construct SDK includes support for **HTML5**    
> Requirements:<br/>
> **Construct 2 or 3**
